# 🎓 EduMarkaz

EduMarkaz — zamonaviy ta’lim platformalarini bir joyda jamlagan Android ilova.
Foydalanuvchilar mashhur online platformalarni ko‘rib chiqishi, taqqoslashi va ular orqali bilim olishi mumkin.

---

## 🚀 Features

* 📚 Top ta’lim platformalar ro‘yxati (Coursera, Udemy, edX va boshqalar)
* 🎥 Video preview (YouTube integratsiya)
* ⚖️ Platformalarni taqqoslash (Compare feature)
* 🔍 Qidiruv va filterlash
* 📱 Zamonaviy UI (Jetpack Compose asosida)
* ⚡ Tez va optimallashtirilgan ishlash

## 📦 Installation (Local run)

1. Repository’ni clone qiling:

```bash
git clone https://github.com/khud0x/Edu-Markaz.git
```

2. Loyihaga kiring:

```bash
cd Edu-Markaz
```

3. Android Studio’da oching

4. Gradle sync bo‘lishini kuting

5. Run ▶️ bosib ishga tushiring
